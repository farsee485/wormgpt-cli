# WormGPT CLI

WormGPT CLI is a sleek command-line interface (CLI) for interacting with LLMs via the OpenRouter API.  
It supports multiple models, automatic language detection, interactive chat, and a terminal-friendly UI.

## Features

- OpenRouter API integration
- Auto language detection (via langdetect)
- Multi-language support: English, Indonesian, Spanish, Arabic, Thai, Portuguese
- Interactive chat session with typing effect
- Stylish CLI UI with colors and banners
- Easy configuration of API key and model
- Config auto-saved in `wormgpt_config.json`
